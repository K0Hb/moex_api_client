## Installation

Adding to a gem:

```ruby
# my-cool-gem.gemspec
Gem::Specification.new do |spec|
  # ...
  spec.add_dependency "moex_api_client"
  # ...
end
```

Or adding to your project:

```ruby
# Gemfile
gem "moex_api_client"
```

### Supported Ruby versions

- Ruby (MRI) >= 2.7.0
- JRuby >= 9.3.0
